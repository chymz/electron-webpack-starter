# Electron/Webpack starter
Personnal starter to build electron apps.

## Features
- Webpack with Babel & Sass (with HMR activated)
- Electron packager
- ASAR builder & Simple ASAR updater
- Branches :
  - `vue` : Starter for VueJS app
  - `phaser` : Starter for a Phaser game
